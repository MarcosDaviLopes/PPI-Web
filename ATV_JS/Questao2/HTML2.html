Window

Window é uma representação de uma janela do navegador onde contem alguns recursos que auxiliam no desenvolvimento do site.

Document

Faz referência aos documentos guardados e que podem ser chamados para modificar o documento HTML

Location

Traz coisas em relação da localização dos documentos e referências

History

Essas variáveis são chamadas de objetos de host do navegador e as mais comuns são: window, document, location e history. window.alert(): exibe uma caixa de diálogo com uma mensagem. document.getElementById(): retorna um elemento HTML com um determinado ID. location.href: retorna a URL completa do documento history.back(): navega para a página anterior no histórico de navegação.
